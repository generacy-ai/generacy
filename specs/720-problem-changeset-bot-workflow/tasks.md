  - `set -euo pipefail` at top.
  - Capture `BASE` / `HEAD` from `${{ github.event.pull_request.base.sha }}` / `head.sha`.
  - **Path-scoped guard**: if no diffed file matches `^packages/[^/]+/src/`, log `No publishable-package source files changed; skipping changeset check.` and `exit 0`.
  - **Test-only short-circuit**: strip `*.test.ts(x)`, `*.spec.ts(x)`, and `/__tests__/` paths; if the in-scope remainder is empty, log `Only test files changed under packages/*/src/; skipping changeset check.` and `exit 0`.
  - **Diff-based changeset detection**: use `git diff --name-only --diff-filter=A "$BASE" "$HEAD" -- '.changeset/*.md'` (excluding `README.md`). The changeset must be **added** in this PR — not merely present on disk — to close the cross-PR leak from incident #707–#717.
  - **Block on miss**: emit the `::error::` line plus follow-up advisory lines from the contract, then `exit 1`.
  - **Pass log**: `Changeset found in PR diff — ready for release.`
