# Data Model: Disambiguate `GENERACY_CLOUD_URL`

**Issue**: #549 | **Branch**: `549-problem-single-env-var`

## Core Type Changes

### 1. `CloudUrlsSchema` (NEW)

```ts
// packages/generacy/src/cli/commands/launch/types.ts

const CloudUrlsSchema = z.object({
  apiUrl: z.string().url(),    // HTTP REST base (e.g., https://api-staging.generacy.ai)
  appUrl: z.string().url(),    // Dashboard/browser (e.g., https://staging.generacy.ai)
  relayUrl: z.string().url(),  // WebSocket relay, fully qualified with ?projectId=
});

export type CloudUrls = z.infer<typeof CloudUrlsSchema>;
```

### 2. `LaunchConfigSchema` (MODIFIED)

```ts
// packages/generacy/src/cli/commands/launch/types.ts

export const LaunchConfigSchema = z.object({
  projectId: z.string().min(1),
  projectName: z.string().min(1),
  variant: z.string().min(1),
  channel: z.enum(['stable', 'preview']).optional(),
  cloudUrl: z.string().url(),                    // DEPRECATED: alias for cloud.appUrl
  clusterId: z.string().min(1),
  imageTag: z.string().min(1),
  orgId: z.string().min(1),
  repos: z.object({
    primary: z.string().min(1),
    dev: z.array(z.string()).optional(),
    clone: z.array(z.string()).optional(),
  }),
  cloud: CloudUrlsSchema.optional(),             // NEW: explicit URL triplet
});
```

### 3. `RegistryEntrySchema` (UNCHANGED)

```ts
// packages/generacy/src/cli/commands/cluster/registry.ts
// NO CHANGES — persisted data, field stays `cloudUrl`

export const RegistryEntrySchema = z.object({
  clusterId: z.string().nullable(),
  name: z.string(),
  path: z.string(),
  composePath: z.string(),
  variant: z.enum(['cluster-base', 'cluster-microservices']).default('cluster-base'),
  channel: z.enum(['stable', 'preview']).default('stable'),
  cloudUrl: z.string().nullable(),    // Stores cloud.appUrl ?? cloudUrl
  lastSeen: z.string().datetime(),
  createdAt: z.string().datetime(),
  managementEndpoint: z.string().optional(),
});
```

## Environment Variable Mapping

### Written to cluster `.env` by scaffolder

| Env Var | Source (new cloud) | Source (old cloud, fallback) |
|---------|-------------------|------------------------------|
| `GENERACY_API_URL` | `LaunchConfig.cloud.apiUrl` | `LaunchConfig.cloudUrl` |
| `GENERACY_RELAY_URL` | `LaunchConfig.cloud.relayUrl` | `deriveRelayUrl(cloudUrl, projectId)` |

### NOT written to cluster `.env`

| Env Var | Reason |
|---------|--------|
| `GENERACY_APP_URL` | No cluster-side consumer (Resolved Q5 = C) |
| `GENERACY_CLOUD_URL` | Dropped from generated `.env` (replaced by above) |

### Reader fallback chains

| Reader | Primary | Fallback | Final default |
|--------|---------|----------|---------------|
| CLI (`resolveApiUrl`) | `GENERACY_API_URL` | `GENERACY_CLOUD_URL` + deprecation log | `https://api.generacy.ai` |
| Orchestrator activation | `GENERACY_API_URL` | `GENERACY_CLOUD_URL` + deprecation log | None (config not set) |
| Orchestrator relay | `GENERACY_RELAY_URL` | `GENERACY_CLOUD_URL` + deprecation log | Derived from `GENERACY_CHANNEL` |
| Cluster-relay | `GENERACY_RELAY_URL` (via orchestrator config) | `GENERACY_CLOUD_URL` (via orchestrator config) | None |

## Validation Rules

- `cloud.apiUrl`: Must be a valid URL (`z.string().url()`), expected `https://` scheme
- `cloud.appUrl`: Must be a valid URL, expected `https://` scheme
- `cloud.relayUrl`: Must be a valid URL, expected `wss://` scheme, should include `?projectId=`
- `cloud` object: Optional on `LaunchConfig` — absence triggers fallback to `cloudUrl` + derivation
- `cloudUrl`: Required on `LaunchConfig` (kept for backward compat during deprecation window)

## Relationships

```
LaunchConfig (cloud response)
├── cloudUrl (string, deprecated)
└── cloud? (CloudUrls, optional)
    ├── apiUrl → written to .env as GENERACY_API_URL
    ├── appUrl → stored in registry as cloudUrl field
    └── relayUrl → written to .env as GENERACY_RELAY_URL

RegistryEntry (persisted in ~/.generacy/clusters.json)
└── cloudUrl ← cloud.appUrl ?? LaunchConfig.cloudUrl

Cluster .env (generated by scaffolder)
├── GENERACY_API_URL ← cloud.apiUrl ?? LaunchConfig.cloudUrl
└── GENERACY_RELAY_URL ← cloud.relayUrl ?? deriveRelayUrl(cloudUrl, projectId)
```
