### Event Streaming

Listen for GitHub events:
- `issues.opened` - New issue created
- `issues.assigned` - Issue assigned to agent
- `issues.labeled` - Label added (workflow triggers)
- `issue_comment.created` - Comment added
- `pull_request.opened` - PR linked to issue

### Workflow Triggers

| Event | Action |
|-------|--------|
| Issue assigned to agent account | Queue for processing |
| Label `autodev:ready` added | Start workflow |
| Comment `@agent continue` | Resume paused workflow |
