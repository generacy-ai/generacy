# Feature Specification: Verify Generacy extension integration with generacy-cloud API

**Branch**: `144-verify-generacy-extension-integration` | **Date**: 2026-01-23 | **Status**: Draft

## Summary

## Overview

The Generacy VS Code extension has cloud features (org dashboard, workflow queue, publishing) that need to be verified against the generacy-cloud API running locally.

## Current State

The extension has cloud views for:
- Organization Dashboard
- Workflow Queue
- Publishing System
- Integration Management

generacy-cloud has full API implementation with:
- `/orgs/*` endpoints
- `/orgs/:id/workflows` endpoints
- `/orgs/:id/queue` endpoints
- Authentication (JWT + GitHub OAuth)
