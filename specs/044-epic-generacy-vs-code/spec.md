# Feature Specification: Epic: Generacy VS Code Extension

**Branch**: `044-epic-generacy-vs-code` | **Date**: 2026-01-21 | **Status**: Draft

## Summary

## Summary

Build the Generacy VS Code Extension - a workflow development IDE for local development and cloud orchestration management.

## Purpose

Generacy extension provides **workflow development capabilities** locally (free) and **organization orchestration** via cloud (paid). It's the bridge between individual development and team-scale AI-driven workflows.

## Pricing Tiers

**Local Mode**: FREE (with generacy.ai account)

**Cloud Mode** (Organization):
| Tier | Price | Limits |
|------|-------|--------|
| Starter | $49/seat/mo (min 3) | 3 concurrent agents, GitHub only |
| Team | $99/seat/mo (min 5) | 10 concurrent agents, all integrations, SSO |
| Enterprise | Custom | Unlimited, compliance, SLA |

## Key Features

### Local Mode (FREE)

#### 1. Workflow Explorer
- Tree view of workflow files (.generacy/*.yaml)
- Create/rename/delete workflows
- Template library

#### 2. Workflow Editor
- YAML-first with intellisense
- Schema validation
- Phase/step visualization
- Variable/secret references

#### 3. Workflow Runner
- Execute workflows locally
- Real-time output streaming
- Environment variable configuration
- Dry-run mode

#### 4. Workflow Debugger
- **Step-through execution**
- **Breakpoints** on phases/steps
- **State inspection** (variables, context, outputs)
- **Replay** from specific step
- Error analysis and recovery

### Cloud Mode (Paid)

#### 5. Organization Dashboard
- Connected org overview
- Member management
- Usage and billing summary

#### 6. Workflow Queue
- View active/pending/completed workflows
- Priority management
- Cancel/retry actions
- Filter by assignee, status, repo

#### 7. Integration Management
- GitHub App connection status
- Issue tracker connections
- CI/CD integrations
- Webhook configuration

#### 8. Publishing
- **Develop → Debug → Publish** pattern
- Push local workflows to cloud
- Version management
- Rollback capability
