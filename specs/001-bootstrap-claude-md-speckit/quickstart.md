# Quickstart: Generacy Repository Setup

## Overview

This guide explains how to use the AI-assisted development infrastructure after bootstrap is complete.

## Prerequisites

- Node.js 20+
- Claude Code CLI installed
- Access to claude-plugins repository

## Installation

No installation required for bootstrap artifacts. They are static configuration files.

## Usage

### Creating New Features

```bash
# Start a new feature specification
/speckit:specify "Feature description"

# Generate implementation plan
/speckit:plan

# Generate task list
/speckit:tasks

# Implement tasks
/speckit:implement
```

### MCP Servers

The configured MCP servers are automatically available:

| Server | Usage |
|--------|-------|
| Context7 | Use for documentation lookup: "What is X?" |
| Playwright | Use for browser automation testing |

### Template Locations

After bootstrap, templates are available at:

```
.specify/templates/
├── spec-template.md      # Used by /speckit:specify
├── plan-template.md      # Used by /speckit:plan
├── tasks-template.md     # Used by /speckit:tasks
├── checklist-template.md # Used by /speckit:checklist
└── agent-file-template.md # Used by update_agent MCP
```

### CLAUDE.md Updates

The `CLAUDE.md` file is automatically updated when:
- New features complete the planning phase
- `update_agent` MCP tool is called

Manual additions should be placed between:
```markdown
<!-- MANUAL ADDITIONS START -->
Your custom content here
<!-- MANUAL ADDITIONS END -->
```

## Troubleshooting

### MCP Server Not Found

If MCP servers fail to start:
1. Ensure Node.js 20+ is installed
2. Check internet connectivity (npx downloads packages)
3. Verify `.mcp.json` is valid JSON

### Template Not Found

If speckit commands fail to find templates:
1. Verify `.specify/templates/` directory exists
2. Check all 5 template files are present
3. Ensure templates are readable markdown files

### CLAUDE.md Not Updating

If CLAUDE.md doesn't update after planning:
1. Run `update_agent` MCP tool manually
2. Check plan.md contains required sections
3. Verify CLAUDE.md is writable
