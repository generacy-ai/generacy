## Tasks

### 1. Create CLAUDE.md

Create a `CLAUDE.md` file at the repository root with:
- Project overview and purpose (orchestration layer connecting Agency and Humancy)
- Active technologies and dependencies
- Project structure documentation (monorepo with orchestrator, workers, plugins)
- Build/test commands
- Code style guidelines
- Manual additions section for repo-specific guidance

Reference: Use the format from `claude-plugins` repo as a template, adapting for this monorepo's structure.

### 2. Set up .speckit templates

Create `.specify/templates/` directory with the following templates copied from `claude-plugins`:
- `spec-template.md` - Feature specification template
- `plan-template.md` - Implementation plan template
- `tasks-template.md` - Task list template
- `checklist-template.md` - Quality checklist template
- `agent-file-template.md` - CLAUDE.md generation template

### 3. Configure .mcp.json

Create `.mcp.json` at the repository root to configure MCP servers for development:

**Recommended servers:**
- **Context7** - Documentation and code context lookup
- **Playwright** - Browser automation for testing web console
- Consider Redis/Docker related MCP servers if available
