# Data Model: Template Config Format Support

## Core Entities

### TemplateConfig (new — input format)

Represents the config format generated by cluster-templates.

```typescript
interface TemplateConfig {
  project?: {
    org_name?: string;
    // Other fields (id, name, org_id) passed through but unused
    [key: string]: unknown;
  };
  repos: {
    primary: string;       // "owner/repo" or "github.com/owner/repo"
    dev: string[];         // Optional, defaults to []
    clone: string[];       // Optional, defaults to []
  };
}
```

### WorkspaceConfig (existing — output format)

The existing workspace config type. No changes.

```typescript
interface WorkspaceConfig {
  org: string;             // GitHub organization
  branch: string;          // Default branch (default: "develop")
  repos: WorkspaceRepo[];  // At least 1 repo
}

interface WorkspaceRepo {
  name: string;            // Bare repo name (min 1 char)
  monitor: boolean;        // Default: true
}
```

## Conversion Mapping

```
Template Format                    →  WorkspaceConfig
─────────────────────────────────────────────────────────
repos.primary ("owner/repo")       →  org: "owner"
                                      repos[0]: { name: "repo", monitor: true }

repos.dev[] ("owner/repo" each)    →  repos[1..n]: { name: "repo", monitor: true }

repos.clone[] ("owner/repo" each)  →  repos[n+1..m]: { name: "repo", monitor: false }

(not mapped)                       →  branch: "develop" (schema default)
```

## Zod Schemas

### TemplateReposSchema

```typescript
const TemplateReposSchema = z.object({
  primary: z.string().min(1),
  dev: z.array(z.string().min(1)).optional().default([]),
  clone: z.array(z.string().min(1)).optional().default([]),
});
```

### TemplateConfigSchema

```typescript
const TemplateConfigSchema = z.object({
  project: z.object({
    org_name: z.string().optional(),
  }).passthrough().optional(),
  repos: TemplateReposSchema,
});
```

## Validation Rules

| Field | Rule | Error behavior |
|-------|------|---------------|
| `repos` | Must be present | Returns null (not detected as template format) |
| `repos.primary` | Must be non-empty string | Throws (template format detected, validation error) |
| `repos.primary` format | Must be parseable by `parseRepoInput()` | Throws (invalid repo format) |
| `repos.dev[]` | Each entry must be non-empty string | Throws |
| `repos.clone[]` | Each entry must be non-empty string | Throws |
| `project` | Optional, not required for conversion | — |
| `project.org_name` | Used as `defaultOrg` fallback for bare repo names | — |
