# Quickstart: Agent Invocation Abstraction

## Installation

The agent abstraction is part of the generacy core package:

```bash
npm install generacy
```

## Basic Usage

### 1. Create and Register an Agent

```typescript
import {
  AgentRegistry,
  ClaudeCodeInvoker,
} from 'generacy';

// Create registry
const registry = new AgentRegistry();

// Register Claude Code agent
const claudeCode = new ClaudeCodeInvoker();
registry.register(claudeCode);

// Set as default
registry.setDefault('claude-code');
```

### 2. Invoke a Command

```typescript
const result = await registry.getDefault().invoke({
  command: '/speckit:specify',
  context: {
    workingDirectory: '/path/to/project',
    issueNumber: 42,
    branch: 'feature/42_my-feature',
  },
  timeout: 300000, // 5 minutes
});

if (result.success) {
  console.log('Output:', result.output);
  console.log('Duration:', result.duration, 'ms');
} else {
  console.error('Failed:', result.error?.message);
}
```

### 3. Check Agent Availability

```typescript
const agent = registry.get('claude-code');

if (await agent?.isAvailable()) {
  await agent.initialize();
  // Ready to invoke
} else {
  console.error('Claude Code CLI not available');
}
```

### 4. Pass Mode Through Context

```typescript
const result = await agent.invoke({
  command: '/speckit:plan',
  context: {
    workingDirectory: process.cwd(),
    mode: 'autodev',  // Mode flows through context
    environment: {
      CUSTOM_VAR: 'value',
    },
  },
});
```

### 5. Access Tool Call Records

```typescript
const result = await agent.invoke({
  command: '/autodev:continue',
  context: { workingDirectory: '.' },
});

if (result.toolCalls?.length) {
  for (const call of result.toolCalls) {
    console.log(`Tool: ${call.toolName}`);
    console.log(`Success: ${call.success}`);
    console.log(`Duration: ${call.duration}ms`);
    if (!call.success && call.errorMessage) {
      console.error(`Error: ${call.errorMessage}`);
    }
  }
}
```

## Error Handling

### Infrastructure Errors (Thrown)

These errors indicate systemic problems:

```typescript
import {
  AgentUnavailableError,
  AgentInitializationError,
  AgentNotFoundError,
  DefaultAgentNotConfiguredError,
} from 'generacy';

try {
  await agent.initialize();
} catch (error) {
  if (error instanceof AgentUnavailableError) {
    console.error('Agent CLI not found. Is it installed?');
  } else if (error instanceof AgentInitializationError) {
    console.error('Agent failed to initialize:', error.message);
  }
}
```

### Invocation Errors (Returned)

These errors are expected workflow outcomes:

```typescript
const result = await agent.invoke(config);

if (!result.success) {
  switch (result.error?.code) {
    case 'TIMEOUT':
      console.error('Invocation timed out');
      break;
    case 'COMMAND_FAILED':
      console.error('Command exited with code:', result.exitCode);
      break;
    case 'AGENT_ERROR':
      console.error('Agent error:', result.error.message);
      break;
  }
}
```

## Troubleshooting

### "Agent not available"

1. Verify the CLI is installed: `which claude`
2. Check PATH includes the CLI location
3. Verify CLI permissions: `ls -la $(which claude)`

### Timeout Issues

```typescript
// Increase timeout for long operations
const result = await agent.invoke({
  command: '/speckit:implement',
  context: { workingDirectory: '.' },
  timeout: 600000, // 10 minutes
});
```

### Missing Tool Calls

Tool call parsing is best-effort. If `toolCalls` is empty:
- The agent may not output structured tool call data
- The output format may have changed
- This doesn't affect invocation success
