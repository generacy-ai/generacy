# T003 Quick Guide: Register VS Code Marketplace Publisher

**Duration**: ~5-10 minutes
**Type**: Manual Web Interface Task

---

## Quick Steps

1. **Navigate**: https://marketplace.visualstudio.com/manage
2. **Sign In**: Use chris@generacy.ai Microsoft account
3. **Create Publisher**:
   - Publisher ID: `generacy-ai` (or alternate if unavailable)
   - Display Name: `Generacy`
   - Description: `AI-powered development workflow tooling`
4. **Link Azure DevOps**: Select `generacy-ai` organization
5. **Verify Email**: Check inbox if prompted
6. **Record**: Publisher ID and profile URL in `T003-progress.md`
