## Summary

Add a single env var line (`GENERACY_BOOTSTRAP_MODE=wizard`) to the `.env` output of `scaffoldEnvFile()` in the shared cluster scaffolder. This tells cluster-base entrypoint scripts to defer repo cloning until the bootstrap wizard completes and credentials are available. The change is unconditional — all callers (launch and deploy) emit wizard mode.
