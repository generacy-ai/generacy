# Quickstart: Create Workflow Command

## Usage

1. Open a workspace folder in VS Code
2. Open Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
3. Type "Generacy: Create Workflow"
4. Select a template from the list
5. Enter a name for your workflow
6. The workflow file opens in the editor

## Available Templates

| Template | Description | Best For |
|----------|-------------|----------|
| Basic | Single phase with simple steps | Getting started |
| Multi-Phase | Setup, build, deploy phases | CI/CD workflows |
| With Triggers | Webhook and schedule triggers | Automated workflows |

## File Location

Workflows are created at:
```
<workspace>/.generacy/workflows/<name>.yaml
```

## Workflow Name Rules

- Must start with a letter
- Can contain: letters, numbers, hyphens (`-`), underscores (`_`)
- Maximum 64 characters

## Error Messages

| Message | Solution |
|---------|----------|
| "No workspace folder open" | Open a folder first (`File > Open Folder`) |
| "Workflow already exists" | Choose a different name |
