### T003 [DONE] [US1] Create tar extraction utility
**File**: `packages/generacy/src/cli/commands/init/tar-utils.ts` (new)
- Implement `extractTarGz(buffer: Buffer, filter: (path: string) => boolean): Promise<Map<string, string>>`
- Use `node:zlib.createGunzip()` + streaming tar parsing
- Either use `tar-stream` package (lightweight, 7KB) or implement minimal ustar header parser (~80 lines)
- Handle 512-byte tar headers: name, size, type flag
- Skip non-file entries (directories, symlinks)
- Decode file content as UTF-8 strings
- Apply filter function to select only matching paths

### T004 [DONE] [P] [US1] Create template fetcher module
**File**: `packages/generacy/src/cli/commands/init/template-fetcher.ts` (new)
- Implement `fetchClusterTemplates(options: FetchOptions): Promise<Map<string, string>>`
- `FetchOptions`: `{ variant: ClusterVariant, ref?: string, token?: string | null, refreshCache?: boolean }`
- Cache logic:
  - Check `~/.generacy/template-cache/{ref}/{variant}/` for cached files
  - If cached and `!refreshCache`, read from cache via `node:fs` and return `Map<path, content>`
  - Create cache dirs with `{ recursive: true }`
- Fetch logic:
  - `GET https://api.github.com/repos/generacy-ai/cluster-templates/tarball/{ref}`
  - Include `Authorization: Bearer {token}` header if token provided
  - Include `Accept: application/vnd.github+json` header
  - Call `extractTarGz()` from `tar-utils.ts` with filter for `{variant}/` directory
  - Map archive paths: strip top-level dir + variant prefix → prepend `.devcontainer/`
    (e.g., `{hash}/standard/Dockerfile` → `.devcontainer/Dockerfile`)
  - Write extracted files to cache directory
  - Return `Map<targetPath, fileContent>`
- Error handling:
  - HTTP 404: "Template ref '{ref}' not found in cluster-templates repository"
  - HTTP 403/401: "Authentication required or rate limited — provide a GitHub token"
  - Network error: "Failed to fetch cluster templates — check your network connection"
  - Fail fast, no retry

- Conditional multi-repo section (`GENERACY_POLL_INTERVAL_MS`, `GENERACY_WORKER_COUNT`) when `devRepos.length > 0`
- Include standard env vars: `GENERACY_PROJECT_ID`, `GENERACY_BASE_BRANCH`, `GENERACY_AGENT`, `GITHUB_TOKEN`

#### `.generacy/.gitignore`
- Static string constant (no variable interpolation needed)
- Contents: `generacy.env`, `*.log`, `.cache/`

#### `.vscode/extensions.json`
- Define `GENERACY_EXTENSIONS = ['generacy-ai.agency', 'generacy-ai.generacy']`
- Implement merge logic accepting optional `existingContent?: string`:
