## Summary

Add UI components for monitoring and managing the agent orchestration system.

## Context

Generacy (VS Code extension + web UI) serves as the human interface for the agent orchestration system. Users need visibility into:
- Work queue status
- Agent pool health
- Issue dispatch/routing
- Progress tracking

## Proposed Features

### Dashboard View
- Work queue visualization (pending, in-progress, completed)
- Agent status cards (available, busy, offline)
- Recent activity feed

### Work Management
- View queued items
- Priority adjustment
- Manual dispatch/reassignment
- Cancel/retry failed work

### Agent Monitoring
- Agent health status
- Current assignments
- Performance metrics
- Logs/output access

## Technical Considerations

- Real-time updates via WebSocket or polling
- API integration with orchestrator service
- May need corresponding VS Code extension views
