# Implementation Plan: Remove cloud-side credential storage and OIDC code

**Feature**: Remove obsolete `generacy-cloud` backend and session-token auth endpoints from credhelper-daemon
**Branch**: `488-context-credentials`
**Status**: Complete

## Summary

Pure deletion task: remove the `generacy-cloud` backend, `SessionTokenStore`, `JwtParser`, and the three `/auth/session-token` control-server routes from `packages/credhelper-daemon`. Update the backend factory error message to reference `cluster-local` and `env` as valid backend types. Remove the `jose` dependency (sole consumer is `jwt-parser.ts`).

## Technical Context

**Language/Version**: TypeScript 5.x (Node.js)
**Primary Dependencies**: Node.js built-in `http` module, `jose` (being removed)
**Storage**: Filesystem (session-token file being removed)
**Testing**: Vitest (`pnpm test`)
**Target Platform**: Linux (Unix socket daemon)
**Project Type**: Monorepo package (`packages/credhelper-daemon`)

## Execution Order

The work naturally splits into three phases executed sequentially:

### Phase A: Delete source files (no compilation needed)
1. Delete `src/auth/jwt-parser.ts`
2. Delete `src/auth/session-token-store.ts`
3. Delete `src/backends/generacy-cloud-backend.ts`

### Phase B: Fix all references to deleted modules
4. Modify `src/backends/factory.ts` — remove `generacy-cloud` case, remove `SessionTokenStore` param, update error message
5. Modify `src/control-server.ts` — remove 3 auth routes, remove `SessionTokenStore` constructor param
6. Modify `src/types.ts` — remove `SessionTokenStore` and `generacyCloudApiUrl` from `DaemonConfig`
7. Modify `src/daemon.ts` — remove `sessionTokenStore.loadFromDisk()` and wiring
8. Modify `bin/credhelper-daemon.ts` — remove `JwtParser`/`SessionTokenStore` imports and instantiation

### Phase C: Delete test files and fix remaining test references
9. Delete `__tests__/auth/jwt-parser.test.ts`
10. Delete `__tests__/auth/session-token-store.test.ts`
11. Delete `__tests__/backends/generacy-cloud-backend.test.ts`
12. Delete `__tests__/integration/session-token-flow.test.ts`
13. Modify `__tests__/backends/factory.test.ts` — remove `generacy-cloud` tests and `SessionTokenStore` mock
14. Modify `__tests__/control-server.test.ts` — remove auth endpoint test cases and `SessionTokenStore` mock
15. Modify `__tests__/integration/config-loading.test.ts` — remove `SessionTokenStore`/`JwtParser` usage

### Phase D: Cleanup and verify
16. Remove `jose` from `package.json` dependencies
17. Run `pnpm install` to update lockfile
18. Run `pnpm build` to verify compilation
19. Run `pnpm test` to verify all tests pass
20. Run repo-wide grep to confirm no dangling imports
