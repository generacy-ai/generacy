# Implementation Plan: Add `--cloud-url` flag to CLI commands

## Summary

Add a `--cloud-url <url>` option to `generacy launch` and `generacy deploy` so that non-production cloud environments can be targeted without shell-specific environment variable syntax. This enables the cloud UI's copy-paste onboarding flow to embed the cloud URL directly in the command.

- **Language**: TypeScript (ESM)

- **CLI framework**: Commander.js
- **Validation**: Zod
- **Package**: `packages/generacy/` (`@generacy-ai/generacy`)
