# Quickstart: GitHub OAuth Authentication

## Overview

The Generacy VS Code extension uses GitHub OAuth for authentication, with progressive tiers unlocking features.

## Usage

### Signing In

1. Open Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
2. Run `Generacy: Sign In`
3. Browser opens to GitHub authorization page
4. Approve the OAuth request
5. VS Code automatically receives the callback

### Signing Out

1. Open Command Palette
2. Run `Generacy: Sign Out`
3. Confirm in the modal dialog

### Viewing Account

1. Open Command Palette
2. Run `Generacy: Show Account`
3. View tier, organization, and available actions

## API Usage

### Check Authentication State

```typescript
import { getAuthService, AuthTier } from './api/auth';

const authService = getAuthService();

// Check if authenticated
if (authService.isAuthenticated()) {
  const user = authService.getUser();
  console.log(`Logged in as ${user?.username}`);
}

// Check tier
if (authService.hasMinimumTier(AuthTier.Organization)) {
  // Enable cloud features
}
```

### Listen for Authentication Changes

```typescript
import { getAuthService } from './api/auth';

const authService = getAuthService();

const disposable = authService.onDidChange((event) => {
  console.log(`Auth changed: ${event.reason}`);
  console.log(`Authenticated: ${event.newState.isAuthenticated}`);
});

// Clean up when done
disposable.dispose();
```

### Get Access Token

```typescript
import { getAuthService } from './api/auth';

const authService = getAuthService();

// Token is automatically refreshed if near expiration
const token = await authService.getAccessToken();
if (token) {
  // Use token for API requests
  const response = await fetch('https://api.generacy.ai/v1/resource', {
    headers: {
      'Authorization': `Bearer ${token}`
    }
  });
}
```

### Guard Features by Tier

```typescript
import { requiresTier, AuthTier } from './commands/cloud';

function myCloudFeature() {
  if (!requiresTier(AuthTier.Organization, 'Cloud Sync')) {
    return; // User was prompted to upgrade
  }

  // Proceed with cloud feature
}
```

| `generacy.login` | Start OAuth login flow |
| `generacy.logout` | Sign out of current account |

## Context Keys

For `when` clauses in package.json:

| Key | Description |
|-----|-------------|
| `generacy.isAuthenticated` | `true` when user is logged in |

Example in package.json:
```json
{
  "command": "generacy.cloudSync",
  "when": "generacy.isAuthenticated"
}
```

## Troubleshooting

### Login doesn't complete

1. Check that the browser opened correctly
2. Ensure you approved the OAuth request
3. Try signing out and signing in again
4. Check VS Code Output > Generacy for error logs

### Token expired

The extension automatically refreshes tokens. If you see "session expired":
1. Token refresh failed (network issue or revoked token)
2. Click "Login" in the notification
3. Complete OAuth flow again

### Wrong account

1. Run `Generacy: Sign Out`
2. Run `Generacy: Sign In`
3. Authorize with the correct GitHub account
