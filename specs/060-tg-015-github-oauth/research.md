# Research: GitHub OAuth Authentication

## Technology Decisions

### 1. VS Code URI Handler for OAuth Callback

**Decision**: Use `vscode.window.registerUriHandler` for OAuth callback

**Rationale**:
- Native VS Code pattern for OAuth flows
- No need for local HTTP server
- Works across all platforms
- Deep link format: `vscode://generacy-ai.generacy-extension/auth/callback`

**Alternatives Considered**:
- Local HTTP server (rejected: port conflicts, firewall issues)
- Manual token entry (rejected: poor UX)

### 2. SecretStorage for Token Persistence

**Decision**: Use VS Code SecretStorage API

**Rationale**:
- Stores in OS keychain (Keychain on macOS, Credential Vault on Windows, libsecret on Linux)
- Encrypted at rest
- Survives extension updates
- VS Code recommended approach

**Alternatives Considered**:
- Workspace state (rejected: not encrypted)
- External file (rejected: security risk)
- globalState (rejected: not suitable for secrets)

### 3. Singleton Service Pattern

**Decision**: AuthService as singleton with initialization

**Rationale**:
- Single source of truth for auth state
- Easy access from any component via `getAuthService()`
- Centralized event distribution

**Alternatives Considered**:
- Context-based injection (rejected: adds complexity for VS Code extensions)
- Module-level state (rejected: harder to test)

### 4. Token Refresh Strategy

**Decision**: Proactive refresh 5 minutes before expiration

**Rationale**:
- Prevents failed requests due to expired tokens
- Background refresh doesn't interrupt user
- Falls back to re-authentication if refresh fails

**Implementation**:
- Timer scheduled after successful auth
- On refresh failure, clear auth and notify user

## Implementation Patterns

### CSRF Protection

State parameter flow:
1. Generate 32-byte random state
2. Store in SecretStorage before redirect
3. Verify on callback
4. Clear after use

### Error Handling

Using extension's `GeneracyError` class with `ErrorCode.AuthFailed`:
- Wrapped fetch errors
- OAuth error responses
- State mismatch errors

## API Contract

### Token Exchange

**Request**: `POST /auth/token`
```json
{
  "code": "authorization_code",
  "redirect_uri": "vscode://generacy-ai.generacy-extension/auth/callback"
}
```

**Response**:
```json
{
  "access_token": "...",
  "refresh_token": "...",
  "expires_in": 3600,
  "user": {
    "id": "...",
    "username": "...",
    "display_name": "...",
    "email": "...",
    "avatar_url": "...",
    "tier": "free|organization",
    "organization_id": "...",
    "organization_name": "..."
  }
}
```

### Token Refresh

**Request**: `POST /auth/refresh`
```json
{
  "refresh_token": "..."
}
```

**Response**:
```json
{
  "access_token": "...",
  "refresh_token": "...",
  "expires_in": 3600
}
```
