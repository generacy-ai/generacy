## 5.3 — Generacy VS Code extension — generacy.env configuration helper

### Tasks
- Command: "Generacy: Configure Environment"
- Checks if `.generacy/generacy.env` exists
- If not, copies from `generacy.env.template`
- Prompts user for required values:
  - `GITHUB_TOKEN` (or guides to `gh auth login`)
  - `ANTHROPIC_API_KEY` (for Claude Code)
  - Optional: `GENERACY_API_KEY` (for cloud features)
- Validates values (token scopes, key format)

### Acceptance Criteria
Extension guides user through env setup. Docker compose can start with configured env.
