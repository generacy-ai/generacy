# Agency Configuration Reference

Configuration reference for the Agency VS Code extension (`agency.config.json`).

:::note Work in Progress

The Agency extension is under active development ([agency#294](https://github.com/generacy-ai/agency/issues/294)). The configuration schema is not yet stable and may change before the extension reaches its first release. A full schema reference will be added here once the extension ships.

:::

## Configuration File

Agency reads configuration from `.agency/config.json` in your project root. The file is auto-generated by the extension on first activation.

```json title=".agency/config.json"
{
  "version": "1.0",
  "project": {
    "name": "my-project"
  }
}
```

## Known Top-Level Fields

The following top-level fields are expected in the initial release, but their exact shapes are subject to change:

| Field | Type | Description |
|-------|------|-------------|
| `version` | `string` | Configuration schema version |
| `project` | `object` | Project metadata |

Additional fields (tools, context providers, plugins) are planned but not yet finalized.
